# SFA time constant (tau_a) fit -- provenance

- Source: raw long-square current-clamp sweeps, Allen synaptic physiology (Campagnola et al., Science 2022), fit for a firing-rate adaptation time constant.
- Database tier: `small`; current version: `{'db_file': 'synphys_r2.1_small.sqlite', 'url': 'https://allen-synphys.s3-us-west-2.amazonaws.com/synphys_r2.1_small.sqlite', 'schema_version': '22', 'release_version': '2.1', 'db_size': 'small'}`.
- Species: mouse; cell types: pyr, pvalb, sst, vip (same CellClass defs as extract_campagnola_matrices.py).
- Experiments sampled: 25 (ranked by typed-cell count); NWB cache: /Users/richner.thomas/ai_synphys_cache.
- Fit model: single  f(t)=f_inf+(f0-f_inf)exp(-t/tau_a), instantaneous rate 1/ISI at ISI midpoints vs time-in-step.
- QC: >= 5 spikes/sweep, R^2 >= 0.5, adaptation >= 0.05, tau in (0.005, 5.0) s (not railed); per-cell median over passing sweeps.
- Pure fit: tau_a is reported for every type (with median adaptation_index as context). Whether a weakly-adapting type (e.g. Pvalb) uses n_a=0 SFA timescales is decided later in the MATLAB Campagnola->parameter mapping, not here.
- Generated by `scripts/aisynphys/fit_sfa_tau.py`.
